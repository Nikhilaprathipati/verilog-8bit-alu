# 8-bit Arithmetic Logic Unit (ALU) in Verilog

## Overview
This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL. 
The ALU performs various arithmetic and logical operations based on an opcode input.

## Features
- 8-bit input operands
- Multiple arithmetic and logical operations
- Combinational ALU design
- Verilog testbench for verification

## Operations Implemented
| Opcode | Operation |
|------|-----------|
|0000| Addition |
|0001| Subtraction |
|0010| AND |
|0011| OR |
|0100| XOR |
|0101| NOT |
|0110| Left Shift |
|0111| Right Shift |

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave

## Project Structure
alu.v        -> ALU design module
alu_tb.v     -> Testbench
waveform.png -> Simulation output
